4인 팀 프로젝트

기간 : 2023.09 ~ 2023.12

다양한 OTT 플랫폼의 콘텐츠를 한 곳에서 관리하고, 시청 페이지에 편리하게 접근하기 위해 개발했습니다. 찜한 콘텐츠의 연관 콘텐츠를 추천받고 관심 콘텐츠를 사용자 편의대로 카테고리를 나눠 분류할 수 있습니다.

    주요 구현 내용

  • 크롬 익스텐션을 통해 OTT 플랫폼 웹 페이지 DOM 인식 및 조작
  • DB 관리 및 연관 콘텐츠 추천 서비스 서버
  • 관리자 페이지

개인 기여

다음 역할을 맡아 프로젝트를 진행했습니다.

  1. 문서 작성 및 파일 관리
  2. 스케줄 관리 및 토의 주도
  3. DB 및 API 설계
  4. 크롬 익스텐션 구현
  5. 추천 시스템 구현

DB 및 API 설계

구현에 앞서 ERD 와 테이블 상세를 준비했고, 어떻게 중복되는 속성없이 테이블을 구성할지 고민했습니다.

프라이머리 키를 잘 선택해야함을 몸소 느꼈고, 적절히 복합키를 선택했습니다. 인덱싱을 활용하여 자주 접근하는 속성에 대한 검색 속도를 높였습니다.

API를 설계하는 과정에서 어떤 종류의 데이터를 주고받아야 하는지 고민했고, 서버와 사용자 화면의 데이터 일관성 문제에 대해 고민했습니다.

외관적으로 버튼 하나를 추가하는 것은 단순하지만, 그 안에서 꽤 많은 종류의 데이터가 오가고 생각해보아야 할 점이 많다는 걸 느끼게 됐습니다.

화면 구현

준비된 시안과 화면 설계를 바탕으로 크롬 익스텐션의 화면을 구현했습니다.

HTML, CSS, JavaScript 를 활용해서 구현했습니다.

    구현 기능

  • 관심 콘텐츠 추가/제거
  • 로그인/로그아웃 기능
  • 신규 카테고리 생성 및 삭제
  • 카테고리별 콘텐츠 분류
  • 연관 콘텐츠 조회
  • OTT 플랫폼별 찜한 콘텐츠 조회

추천 시스템 구현

MovieLens에서 제공하는 데이터를 바탕으로 Matrix Factorization을 변형시켜 구현했습니다.

학습하는 데이터의 종류와 품질에 따라 많은 것이 좌우된다는 것을 알게되었고

섬세한 데이터 전처리의 중요성을 알게 되었습니다.

pytorch 라이브러리와 numpy 의 기본 연산을 활용하여 구현했습니다.
